# Agent Rules

All project rules are defined in [`CLAUDE.md`](CLAUDE.md) at the repository root.
**Read that file before making any changes.** The rules there are authoritative and apply to all contributors and agents.

Key rules (critical minimum if `CLAUDE.md` is not loaded):
- All files under `docs/` subdirectories must use zero-padded three-digit sequential prefixes (`001-`, `002-`, …). List existing files before creating a new one — never guess the next number.
